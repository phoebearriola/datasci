  sapply(1:p, function(k){
    sum(1 / (k:p)) / p
  })

### Interpretation of Part B: Broken-stick vs PCAtest

# The broken-stick and PCAtest methods help in assessing principal components and which ones have significant biological data instead of random noise 

# Broken-stick - compares the eigenvalue of each PCA component to random values. If the eigenvalue is larger than the corresponding number of broken-stick, then it is considered as significant.

# PCA test - utilizes permutations as the data is randomized or shuffled multiple times. If a component presents a variance that is significantly greater than what is expected from random data, then that component is said to be significant.
